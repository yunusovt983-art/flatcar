## What is this?

Flatcar Builder is a declarative factory for building reproducible, immutable Flatcar Linux
from Gentoo source with a verifiable supply-chain. Architecture: Domain-Driven Design
(hexagonal, 14 Rust crates, ports & adapters).

This repository tracks epic progress publicly. The source code lives in a private repository.
